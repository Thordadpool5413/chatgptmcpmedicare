# Data access matrix

This server is intentionally built around free public data and local internal files that you provide.

## Free public data this server can use

1. CMS Data API public datasets
2. CMS Provider Data Catalog public datasets
3. NPPES NPI Registry API
4. Medicare Post Acute Care Utilization Hospice public use file
5. Medicare Inpatient Hospitals by Provider and Service public dataset
6. Hospice Provider Data from Provider Data Catalog
7. Nursing Home Provider Information from Provider Data Catalog
8. A local internal admissions CSV that you export and control

## Data this server does not unlock for free

1. Exact provider to provider referral flows
2. Beneficiary level Medicare claims
3. CCW research identifiable files
4. VRDC data access
5. CASPER or iQIES restricted reports
6. Raw patient level OASIS, HIS, or HOPE assessment files
7. Paid claims analytics from commercial vendors

## Practical strategy

Use public CMS data for opportunity scoring and competitor intelligence. Use your internal admissions CSV for actual referral source truth. The best free workflow is to compare both layers.

Public data answers: where is the opportunity?

Internal data answers: where are we actually winning?

The gap between those answers becomes the account plan.
